written in Python3.
xlrd, smtplib module is needed.

Takes the Email/password list from a .xlsx spreadsheet in current directory (Test_accounts.xlsx).

Email must be in Coloumn A and password in Coloum C.

Will then attempt to login via selenium webdriver (Only gmail supported currently).

Successful attempts will be written to a seperate text file 'Accounts.txt'.
